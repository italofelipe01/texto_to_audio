# texto_to_audio

Texto para Áudio em Português
Este é um programa simples que usa a biblioteca [gTTS] para converter um texto em áudio em português. Ele também usa a biblioteca [os] para executar o comando [ffplay] para reproduzir o arquivo de áudio gerado.

Requisitos
Python 3.6 ou superior
gTTS 2.2.3 ou superior
ffmpeg 4.4 ou superior
Como usar
Clone este repositório ou baixe o arquivo texto_para_audio.py
Instale as dependências com o comando pip install -r requirements.txt
Execute o programa com o comando python texto_para_audio.py
Digite a sua mensagem a ser disponibilizada em áudio e pressione Enter
Aguarde a geração do arquivo audio.mp3 e a reprodução do mesmo
Exemplo:

Insira sua mensagem a ser disponibilizada em audio por favor: Olá, este é um exemplo de texto para áudio em português.
O programa irá gerar e reproduzir um arquivo de áudio com a seguinte mensagem:

Olá, este é um exemplo de texto para áudio em português.
